## Нормальная форма

База приведена к третьей нормальной форме (3NF).

Основные сущности:

- Пользователи
- Роли
- Клиенты
- Заказы
- Состав заказа
- Товары
- Пункты выдачи

Все связи реализованы внешними ключами.
